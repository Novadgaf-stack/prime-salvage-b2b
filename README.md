# Prime Salvage B2B Marketplace

A high-performance B2B platform engineered for the salvage and parts industry. Built by **Prime Assets**, this application facilitates seamless vendor-to-buyer transactions with real-time data synchronization.

## 🚀 Technical Stack
* **Framework:** Next.js (App Router)
* **Backend/Database:** Firebase Firestore
* **Authentication:** Firebase Auth
* **Styling:** Tailwind CSS & Lucide Icons
* **Deployment:** Vercel

## ✨ Key Features
* **Multi-Sided Dashboard:** Dedicated interfaces for buyers and sellers to manage inventory and orders.
* **Real-time Synchronization:** Powered by Firebase to ensure part availability is always up-to-date.
* **Responsive Design:** Mobile-first architecture optimized for use in the field or the office.
* **Secure Auth:** Enterprise-grade security for user data and transaction history.

## 🛠️ Local Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Create a `.env.local` file and add your Firebase credentials.
4. Start the development server: `npm run dev`
